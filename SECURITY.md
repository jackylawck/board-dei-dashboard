# Security Policy

## Supported Versions
Only the latest main branch of this dashboard is supported with security updates.

## Reporting a Vulnerability
As an AI Governance and HR professional, data protection and system integrity are our highest priorities. If you discover any security vulnerabilities, algorithmic bias anomalies, or privacy leakage risks in this dashboard, please DO NOT open a public GitHub issue.

Please report vulnerabilities privately by connecting via:
* **Professional Network:** [LinkedIn Profile (www.linkedin.com/in/jackylawck)](https://www.linkedin.com/in/jackylawck)

We will acknowledge your report within 48 hours and provide a detailed mitigation plan within 7 days.
